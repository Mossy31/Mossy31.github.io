Code Finder

DateEarly server history
StatusExpired — no winner
Total Books11
Search Radius20,000

Overview

Code Finder was one of the earliest events on AnarchySMP. Players were challenged to locate hidden written books scattered across a 20,000 × 20,000 block area around spawn (covering both positive and negative coordinates). The event went largely unnoticed and no player ever collected all the books or claimed the reward.

Still active The books remain hidden in the world to this day. No player has found all 11. If you want the coordinates, DM @𝖇𝖑𝖚𝖊𝖘𝖙ꬽ𝖓𝖊 on Discord.

How It Worked

Rules
· 11 written books were hidden within a 20,000 × 20,000 block radius of spawn
· Each book was numbered (e.g. 1/11, 2/11, …) and contained a single letter
· There was only one copy of each book — first to find it kept it
· Collecting all 11 letters would form a code redeemable in the server store
· The store reward has since been removed, as no one completed the event

Current Status

Unclaimed
As of May 2026, all 11 books are believed to still be in the world — relics from the early days of the server. The store reward tied to the code no longer exists, but the books themselves remain as historical artefacts. Proof of the event's announcement can be found in the Discord announcements channel by searching CODE FINDER.
